# Employee-Management-System
An Employee Management System built using Java Servlets, JDBC, JSP and MySQL. This project demonstrates a backend-focused Java web application following the MVC architecture.

The Employee Management System allows organizations to manage employee records efficiently.
It supports basic CRUD operations such as adding, viewing, updating, and deleting employee details.
